# README – Machine Learning Models

## Overview

This directory contains 45,000 species distribution models trained on virtual species data. Each model is stored as a serialized R object (`.RDS`).

These models were generated during the execution of the script `04_modelling.R`, which implements a modeling workflow adapted from **Valavi et al. (2023)**. The outputs in this folder correspond one-to-one with the spatial predictions saved in the accompanying `maps` directory and serve as the basis for subsequent performance evaluations.

> **Note on file size**: Due to the total size of all models beeing **52.8GB**, only a small selection of example files is included in this repository. The complete set can be generated by running the script `04_modelling.R`.

---

## Model Specifications & Algorithms

All models were trained within a **5-fold (spatial) cross-validation**.

| Algorithm | Package | Tuning & Specifications |
| :--- | :--- | :--- |
| **Random Forest (RF)** | `randomForest` | Tuned over `mtry` (1 to max predictors); `ntree` = 1000. Background points downsampled to match presence points. |
| **Boosted Regression Trees (BRT)** | `gbm` | Tuned grid: `n.trees` (50, 250), `interaction.depth` (1, 5), `shrinkage` (0.001, 0.0001). Background points downweighted. |
| **GAM (Splines)** | `gam` | Method "gamSpline". Predictors standardized (mean=0, SD=1). Background points downweighted. |
| **Maxent** | `ENMeval` | Restricted to linear features (`fc = "L"`) with a regularization multiplier of 1 (`rm = 1`). |
| **Lasso** | `glmnet` | Includes **quadratic feature expansion** for all predictors. Tuned over a grid of `lambda` values. |

## Naming Convention

Files follow a standardized naming convention to identify the specific simulation parameters. The structure is:

`VSXX_FOLDS_MODEL_testDataD_pointsYYY_replicatesZZ.RDS`

### Parameter Reference Table

| Component | Example | Description |
| :--- | :--- | :--- |
| **VSXX** | `VS01` | **Virtual Species ID**: Identifier for the species (VS01–VS10). |
| **FOLDS** | `block1` | **Partitioning Strategy**: The CV method used (`KNNDM`, `random`, `block1`, `block2`, `clusters`). |
| **MODEL** | `BRT` | Modeling algorithm (`BRT`, `RF`, `GAM`, `Lasso`, or `Maxent`) |
| **testDataD** | `testData1` | **Test Fold**: The specific fold ID (1–6) used as the independent testing set. |
| **pointsYYY** | `points40` | **Sample Size**: The number of presence points sampled. |
| **replicatesZZ** | `replicates1` | **Replicate**: The iteration number (1–5). |
